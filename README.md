# Сайт Маргариты Волковой

Статический лендинг преподавателя английского языка: услуги, тарифы, отзывы, FAQ, форма записи.

## Структура проекта

```
├── index.html          # Главная страница
├── 404.html            # Страница ошибки
├── assets/
│   ├── css/
│   │   └── main.css    # Стили
│   ├── js/
│   │   └── main.js     # Меню, модалка, карусель, EmailJS
│   └── images/         # Изображения и иконки
└── pages/
    ├── offer.html      # Публичная оферта (шаблон)
    └── privacy.html    # Политика обработки ПДн (шаблон)
```

Корень репозитория — это публичная директория для хостинга (GitHub Pages, Firebase Hosting и т.п.): пути вида `assets/...` рассчитаны на раздачу из корня.

## Форма и EmailJS

В `assets/js/main.js` заданы публичный ключ и идентификаторы сервиса и шаблона EmailJS. В шаблоне письма в консоли EmailJS добавьте поля **`name`**, **`email`**, **`phone`**, **`interest`**, **`message`** (или сопоставьте их с переменными шаблона).

## Юридические страницы

Тексты в `pages/offer.html` и `pages/privacy.html` — ознакомительные шаблоны; перед публикацией их стоит проверить с юристом.
